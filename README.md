<h3>Deployment Firebase Hosting</h3>

> Build product: `ng building --prod` <br>
> Install the Firebase CLI via npm by running the following command: <br>
> `npm install -g firebase-tools` <br>
> Login: `firebase login` <br>
> Init: `firebase init`
